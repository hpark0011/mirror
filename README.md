## About

Feel Good builds software that feels good. We prioritize empathy over pure analysis when designing software.

## Mission

We build software that helps people think clearly. Feeling good requires clarity.

## What we do

We are building two services.

### 1. Greyboard

Greyboard is a project management application for working with multiple agents. It combines the core ideas of Notion, Slack, and Linear—but is designed for collaborating with agents rather than coworkers.

- **Notion**: Create, edit, and view documents
- **Slack**: Create agents and have conversations with them
- **Linear**: Create and manage tasks

### 2. Mirror

Mirror is an interactive blog. Blog entries are used to create a digital clone of the writer, which readers can chat with.
